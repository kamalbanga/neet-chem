# --- Units of Rate Constant (by order) ---
[KIN_001] Unit of k for zero order reaction :: mol L⁻¹ s⁻¹
[KIN_002] Unit of k for first order reaction :: s⁻¹
[KIN_003] Unit of k for second order reaction :: L mol⁻¹ s⁻¹

# --- Half-Life Formulas ---
[KIN_004] Half-life formula for first order reaction :: 0.693/k
[KIN_005] Half-life formula for zero order reaction :: [A]₀/2k
[KIN_006] When [A]₀ is doubled in zero order reaction, t₁/₂ is :: doubled
[KIN_007] First order t₁/₂ depends on :: k only

# --- First Order Time Calculations ---
[KIN_008] First order: time for 99% completion :: 4.606/k
[KIN_009] First order: t₉₉.₉% = ___ × t₁/₂ :: 10

# --- Arrhenius Plot ---
[KIN_010] Slope of ln k vs 1/T plot :: −Eₐ/R
[KIN_011] Intercept of ln k vs 1/T plot :: ln A
[KIN_012] A in Arrhenius equation is called ___ factor :: frequency (or pre-exponential)
[KIN_013] For every 10°C rise in temperature, rate constant roughly :: doubles

# --- First Order Graphs ---
[KIN_014] Slope of ln[R] vs t plot for first order :: −k

# --- Zero Order Examples ---
[KIN_015] Zero order: NH₃ decomposition on ___ at high pressure :: platinum
[KIN_016] Zero order: HI decomposition on ___ surface :: gold

# --- First Order Examples ---

# --- Order from Experimental Data ---

[KIN_020] Reaction: 50% complete in 2h, 75% complete in 4h. Order is :: first
