# Haloalkanes and Haloarenes — Flashcards
# Source: NCERT

# --- Classification ---
[HALO_001] In allylic halides, halogen is bonded to an sp³ carbon ___ to a C=C bond :: adjacent
[HALO_002] In vinylic halides, halogen is bonded to an sp² carbon of a ___ :: C=C double bond
[HALO_003] In benzylic halides, halogen is bonded to an sp³ carbon attached to ___ :: aromatic ring
[HALO_004] Halogen directly bonded to sp² carbon of an aromatic ring → ___ halide :: aryl
[HALO_005] gem-Dihalides have both halogens on ___ carbon atom :: same
[HALO_006] vic-Dihalides have halogens on ___ carbon atoms :: adjacent

# --- C–X Bond Properties ---
[HALO_007] C–X bond enthalpy order (highest to lowest) :: C–F > C–Cl > C–Br > C–I
[HALO_008] C–X bond length order (shortest to longest) :: C–F < C–Cl < C–Br < C–I

# --- Preparation of Haloalkanes ---
[HALO_009] R–OH + SOCl₂ → R–Cl + ___ + ___ :: SO₂ + HCl
[HALO_010] R–OH + PCl₅ → R–Cl + ___ + ___ :: POCl₃ + HCl
[HALO_011] SOCl₂ is preferred for R–OH → R–Cl because byproducts are ___ :: gases
[HALO_012] Catalyst for 1° and 2° alcohol + HCl → R–Cl :: ZnCl₂

# --- Halogen Exchange Reactions ---
[HALO_013] Finkelstein reaction: R–Cl/R–Br + NaI in ___ → R–I :: dry acetone
[HALO_014] Swarts reaction converts alkyl halides to ___ using metallic fluorides :: alkyl fluorides
[HALO_015] Common metallic fluorides for Swarts reaction :: AgF, SbF₃

# --- Preparation of Haloarenes ---
[HALO_016] Catalyst for electrophilic halogenation of benzene :: anhy. AlCl₃ or FeCl₃
[HALO_017] Iodination of arenes requires an ___ agent (e.g., HNO₃) :: oxidising
[HALO_018] Sandmeyer's reaction: ArN₂⁺ + ___ → ArCl or ArBr :: CuCl or CuBr
[HALO_019] ArN₂⁺ → ArI requires only ___ (no cuprous halide) :: KI
[HALO_020] Friedel-Crafts reaction requires ___ halide (not aryl or vinylic) :: alkyl

# --- Nucleophilic Substitution: SN1 vs SN2 ---
[HALO_021] SN2 reactivity order of alkyl halides :: methyl > 1° > 2° > 3°
[HALO_022] SN1 reactivity order of alkyl halides :: 3° > 2° > 1°
[HALO_023] SN2 is favoured by ___ solvents (e.g., DMF) :: polar aprotic
[HALO_024] SN1 is favoured by ___ solvents (e.g., water, alcohol) :: polar protic
[HALO_025] SN2 of optically active alkyl halide → ___ of configuration :: inversion
[HALO_026] SN1 of optically active alkyl halide → ___ :: racemisation
[HALO_027] For same R group, halide reactivity in both SN1 and SN2 :: R–I > R–Br > R–Cl >> R–F
[HALO_028] Allylic and benzylic halides are highly reactive in SN1 due to ___ stabilisation :: resonance

# --- Ambident Nucleophiles ---
[HALO_029] R–X + KCN → ___ (cyanide or isocyanide?) :: cyanide (R–CN)
[HALO_030] R–X + AgCN → ___ (cyanide or isocyanide?) :: isocyanide (R–NC)
[HALO_031] R–X + KNO₂ → ___ (nitroalkane or alkyl nitrite?) :: alkyl nitrite (R–ONO)
[HALO_032] R–X + AgNO₂ → ___ (nitroalkane or alkyl nitrite?) :: nitroalkane (R–NO₂)

# --- Elimination Reactions ---
[HALO_033] Reagent for dehydrohalogenation of alkyl halides :: alcoholic KOH
[HALO_034] Aqueous KOH + R–X → ___ :: alcohol (substitution)
[HALO_035] Alcoholic KOH + R–X → ___ :: alkene (elimination)
[HALO_036] Saytzeff's rule: preferred alkene has ___ alkyl groups on doubly bonded carbons :: more
[HALO_037] Major product of dehydrohalogenation of 2-bromopentane :: pent-2-ene

# --- Haloarene Reactivity ---
[HALO_038] Haloarenes are ___ reactive than haloalkanes towards nucleophilic substitution :: less
[HALO_039] Haloarene C–X bond has partial ___ character due to resonance :: double bond
[HALO_040] Halogen on benzene is ___ directing in electrophilic substitution :: ortho, para
[HALO_041] Halogen ___ the benzene ring towards electrophilic substitution :: deactivates
[HALO_042] In haloarene electrophilic substitution, ___ effect controls reactivity :: inductive (–I)
[HALO_043] In haloarene electrophilic substitution, ___ effect controls orientation :: resonance (+M)
[HALO_044] –NO₂ at ortho/para positions ___ nucleophilic substitution reactivity of haloarenes :: increases
[HALO_045] Chlorobenzene → phenol requires NaOH(aq) at ___ K and ___ atm :: 623; 300

# --- Grignard Reagent & Metal Reactions ---
[HALO_046] Grignard reagent general formula :: RMgX
[HALO_047] Solvent for Grignard reagent preparation :: dry ether
[HALO_048] Grignard reagent + H₂O → ___ :: R–H
[HALO_049] Wurtz reaction: 2R–X + 2Na (dry ether) → ___ :: R–R
[HALO_050] Wurtz-Fittig reaction: R–X + Ar–X + 2Na (dry ether) → ___ :: Ar–R
[HALO_051] Fittig reaction: 2Ar–X + 2Na (dry ether) → ___ :: Ar–Ar

# --- Stereochemistry ---

[HALO_053] Racemic mixture is denoted by prefix :: (±) or dl

# --- Physical Properties ---
[HALO_054] Boiling point order of alkyl halides (same R) :: RI > RBr > RCl > RF
[HALO_055] Boiling point of isomeric haloalkanes ___ with increased branching :: decreases
[HALO_056] Among dihalobenzene isomers, ___ has the highest melting point :: para

[HALO_058] Chloroform is stored in dark, full bottles to keep out ___ :: air and light

[HALO_060] Freon-12 formula :: CCl₂F₂
[HALO_061] Freons are prepared by ___ reaction :: Swarts
