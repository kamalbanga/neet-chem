# --- Ethers ---

[APE_046] Tertiary alkyl halide + sodium alkoxide gives ___ instead of ether :: alkene (elimination)
[APE_047] Reactivity order of HX with ethers :: HI > HBr > HCl
[APE_048] Anisole + HI → :: phenol + CH₃I
[APE_049] Ether cleavage: when one alkyl group is tertiary, halide formed via ___ mechanism :: SN1
[APE_050] Anisole + Br₂/CH₃COOH (no Lewis acid catalyst) → major product :: p-bromoanisole
[APE_051] Friedel-Crafts acylation of anisole gives product mainly at ___ position :: para
